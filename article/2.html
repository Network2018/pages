VS 2017 离线安装包下载方式

08/03/2017

本文介绍 Microsoft Visual Studio 2017 的离线安装包下载方式。

前言

VS 全称 “Microsoft Visual Studio” ，是微软公司的开发工具套件系列产品。因为需要用这货来编译，然而国内要连到微软的速度实在不敢恭维，最开始尝试在线安装还被报错下载中断多次，遂只好去找离线包。然而 2017 这版他不提供 iso 了，不过还好有一个离线包下载器。

离线

visual studio community 是免费使用的，我们就选这个了。

先下载这个下载器：https://www.visualstudio.com/zh-hans/thank-you-downloading-visual-studio/?sku=Community&rel=15

这个 exe 是 vs 2017 的在线安装器，但也可用它做为 离线下载器。

将这个 exe 重命名为 vs_community.exe，然后在当前路径运行 cmd：

# 全功能
./vs_community.exe --layout "F:\offline-vs2017" --lang zh-CN
## --layout 输出目录
## --lang   下载语言

# 指定部分功能
./vs_community.exe --layout "F:\offline-vs2017" --lang zh-CN --add Microsoft.VisualStudio.Workload.Universal
## 使用 --add 参数，指定下载组件
## 更多组件参数，请参阅 https://docs.microsoft.com/zh-cn/visualstudio/install/workload-component-id-vs-community

运行 cmd 后，会弹出 vs 下载器的命令行窗口并自动进行下载，会有很长的文字刷屏，耐心等待即可。

下载完成后，双击 vs_community.exe 进去选择安装就行了，安装器会自行调用下载的离线缓存进行安装。
