killall：command not found

01/21/2018

本文介绍如何修复 “killall: command not found” 报错。

killall 是我很喜欢用的一个命令。今天在一台小鸡上折腾时却遇到了这个：

root@naive:~# killall php-fpm
-bash: killall: command not found

要修复这个很简单，只需要运行：

# Debian / Ubuntu
apt-get install -y psmisc

# Centos
yum install -y psmisc
