apt-get：there is no public key available

02/12/2018

本文介绍如何修复 “apt-get：there is no public key available” 报错。

在 virmach 小鸡上运行 apt-get 时又遇到了报错 apt-get：there is no public key available，它家的 debian 模板真的是有毒（

要修复这个问题，需要执行以下：

# 先运行 update 让它报出 there is no public key available 的报错信息
apt-get update
# 然后安装和更新 keyring
apt-get install -y debian-keyring debian-archive-keyring
apt-key update
# 最后再次 update，问题解决
apt-get update
